# 🇱🇹 Lietuvių · Тренажер A2 · ПМП

Веб-тренажер литовської мови (рівень A2 / іспит ПМП) для україномовних.

**Відкрити:** https://justdes93.github.io/lietuva_learn/

## Що всередині
- 20 тем (лексика, відмінки, переклад, граматика)
- 5 режимів: картки, переклад (ввід), вибір варіанту, пропуски (відмінки), відповідності
- граматичний довідник (7 відмінків, відмінювання дієслів)
- прогрес зберігається локально в браузері

Один файл — `index.html`, без залежностей.
