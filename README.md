# 🇱🇹 Lietuvių · Тренажер A1–A2 · ПМП

Веб-тренажер литовської мови (рівень A1–A2 / іспит ПМП) для україномовних.

**Відкрити:** https://justdes93.github.io/lietuva_learn/

## Що всередині
- **1100+ слів** у 28 темах (лексика з підручників SDG)
- **9 типів вправ:** Мікс, картки, вибір LT↔UA, переклад вводом (обидва боки), аудіо-диктант, пропуски (відмінки), склади речення, відповідності
- **🎧 Аудіювання** — тексти озвучуються голосом + питання з варіантами A/B/C (як на іспиті)
- **🖼️ Опис картинок** — 24 повсякденні сцени з питаннями-підказками, словником і зразковим описом (10–15 речень)
- Справжня литовська озвучка (Google TTS) + граматичний довідник
- Прогрес зберігається локально в браузері

Один файл — `index.html`, без залежностей.
